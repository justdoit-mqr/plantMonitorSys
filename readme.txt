plantMonitorSys是个人本科毕业设计作品的主程序

本设计推出了一种基于WIFI网络的车间设备监测与控制系统。
硬件上采用德州仪器公司Cortex-A8架构的ARM处理器作为主控芯片，选用了IAC-335X-Kit开发板进行系统设计，外接USB摄像头模块和WIFI无线模块进行视频采集和数据传输。
软件上以嵌入式Linux系统作为软件平台，通过WIFI无线网络技术连接车间的各个设备。将生产数据精确、客观的收集，并对其统计分析，以提升企业的生产管理效率。同时系统使用QT开发上位机软件GUI界面，提供良好的人机交互，供企业管理者对车间设备集中式监测与控制，为打造数字化工厂奠定基础。本设计相较于基于有线网络的监测控制系统，采用WIFI 网络更加便捷、灵活，减小了传统网络布线的实施限制和维护费用，降低了企业的成本投资。

注:本程序最初版本使用Qt4.8.2版本的Qt库进行开发，同时为了实现视频采集与保存，调用了ffmpeg的一些库，这些库交叉编译应用于特定的arm-linux平台，所以当使用其他版本的Qt或交叉编译器编译时该代码时，可能会因为不兼容而编译失败。

为了方便在其他平台上也能成功编译该程序源码，V2.0版本对程序代码结构进行了部分修改与调整。
1.将视频采集与编码保存的功能独立了出来,可以通过pro文件的宏“#DEFINES += VIDEO_CAPTURE 　#DEFINES += VIDEO_ENCODE_SAVE ”控制是否开启该功能
2.该版本使用Qt5编译，替换了几个不再支持的Qt4方法。
3.为了适应PC端，软键盘窗口不再全屏展示
4.调整数据库文件的存放位置
5.其他的一些代码细节的优化
